# Lab 7 Instructions: Personalized Job Application Coach

## 1. Setup Environment
Install dependencies:
```bash
pip install -r requirements.txt
```

## 2. Gradio Demo (`gradio_demo.py`)
A simple app to add two numbers, demonstrating Gradio basics.
```bash
python gradio_demo.py
```

## 3. Simple LLM (`simple_llm.py`)
A script to test Watsonx Llama 3 via terminal.
```bash
python simple_llm.py
```

## 4. LLM Chat (`llm_chat.py`)
A Gradio-based chat interface for the LLM.
```bash
python llm_chat.py
```

## 5. Resume Polisher (`resume_polisher.py`)
An app to polish your resume based on a position.
```bash
python resume_polisher.py
```

## 6. Cover Letter Generator (`cover_letter.py`)
An app to generate a customized cover letter.
```bash
python cover_letter.py
```

## 7. Career Advisor (`career_advisor.py`)
An app to provide career advice based on your resume and a job description.
```bash
python career_advisor.py
```
